This program is very simple to use.

1. Simply run the hw1.py python file
2. It will ask which question you want the answer to.
3. Simply type the question number (for eg. 3 and hit enter)
4. The program will display the answer to the specified question by calculating in realtime
	(no answers stored previously)
5. After displaying the result, the program will ask if you want to continue asking
6. Type y for yes and n for no and hit enter.
7. If entered y, type another question you want the answer to and hit enter. 
8. If entered n, program terminates.
